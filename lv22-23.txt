A program is running automatically at regular intervals from cron, the time-based job scheduler. 
Look in /etc/cron.d/ for the configuration and see what command is being executed.
NOTE: Looking at shell scripts written by other people is a very useful skill. 
The script for this level is intentionally made easy to read. 
If you are having problems understanding what it does, 
try executing it to see the debug information it prints.

Tiếp tục tìm hiểu tại /etc/cron.d
script /cronjob_bandit23.sh:
#!/bin/bash

myname=$(whoami)
mytarget=$(echo I am user $myname | md5sum | cut -d ' ' -f 1)

echo "Copying passwordfile /etc/bandit_pass/$myname to /tmp/$mytarget"

cat /etc/bandit_pass/$myname > /tmp/$mytarget
Script này sẽ băm string "I am user ..." trong đó ... là của biến myname. Vậy thì ta tạo biến tạm thời là bandit23 để lấy pass tương ứng

bandit22@bandit:/etc/cron.d$ myname=bandit23
